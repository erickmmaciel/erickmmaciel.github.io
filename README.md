# erick2m.com

Site institucional da Erick Maciel Marketing LTDA.

Site estático servido via GitHub Pages no domínio `erick2m.com`.

## Stack
- HTML estático puro
- CSS (sem framework)
- Fonts: Inter + Fraunces (Google Fonts)

## Estrutura
- `index.html` — landing page única
- `style.css` — estilos
- `404.html` — página de erro
- `CNAME` — domínio customizado para GitHub Pages

## Deploy
Push em `main` → GitHub Pages publica automaticamente.
